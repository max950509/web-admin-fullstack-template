# 可观测性（Metrics + Tracing）

本项目使用 OpenTelemetry + Prometheus + Grafana + Tempo。

## 目录
- 组件说明
- 启动方式

## 组件说明
- **OpenTelemetry Collector**：统一接收 OTLP 数据并转发。
- **Prometheus**：采集指标。
- **Tempo**：存储与查询 Trace。
- **Grafana**：统一展示指标和链路。

## 启动方式

```bash
docker compose -f deploy/observability/docker-compose.yml up -d
```

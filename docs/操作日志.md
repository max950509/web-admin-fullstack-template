# 操作日志

系统内所有关键操作都会写入审计日志，并带有脱敏策略。

## 1. 记录方式

- 通过 `OperationLogInterceptor` 在全局拦截
- 自动记录：
  - 用户信息
  - action/resource
  - 方法与路径
  - 状态码
  - IP / User-Agent
  - 请求 payload（脱敏后）

## 2. 脱敏策略

内置敏感字段名单：

- `password`、`token`、`authorization`、`captcha` 等

拦截器会递归遍历 payload，将敏感字段替换为 `***`。

## 3. 降噪策略

以下请求不记录日志：

- `OPTIONS` 预检请求
- `/operation-logs` 自身查询

## 4. 日志展示

- 后端支持分页查询
- 前端提供“操作日志”页面
- Dashboard 可展示个人近期日志
